# Code of Conduct

## Purpose

This project is committed to a respectful, professional, and technically focused collaboration environment.

## Expected Behavior

All participants are expected to:

- Communicate respectfully and constructively.
- Focus on technical facts, reproducible behavior, and clear reasoning.
- Assume good intent and ask clarifying questions before escalating disagreements.
- Keep review feedback specific, actionable, and relevant to the change.

## Unacceptable Behavior

The following is not acceptable:

- Harassment, intimidation, or threats
- Personal attacks or insulting language
- Discriminatory or exclusionary conduct
- Repeated disruptive behavior after being asked to stop
- Publishing private information without consent

## Scope

This Code of Conduct applies to:

- Issues, pull requests, and review comments
- Project discussions and related communication channels

## Reporting

If you experience or witness unacceptable behavior, report it to the maintainers with:

- A short description of what happened
- Links or screenshots if available
- Relevant timestamps and context

Reports will be reviewed promptly and handled as confidentially as practical.

## Enforcement

Maintainers are responsible for enforcing this policy and may:

- Request behavioral changes
- Remove inappropriate content
- Temporarily or permanently restrict participation

Actions are based on severity and pattern of behavior.
