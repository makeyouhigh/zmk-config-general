# 한국어 레이아웃 가이드

이 문서는 ZMK 기반 키보드에서 한국어 입력 방식을 선택하고 운용하기 위한 기준을 정의한다.
이 저장소는 한국어 입력을 펌웨어에서 처리하지 않는다.

---

## 기본 원칙

### OS 위임 입력 모델

이 저장소는 **OS 위임(OS-delegated) 입력 모델**을 강제한다.

- 펌웨어는 표준 HID 키코드만 전송한다.
- 한글 조합, 자모 처리, 입력 모드는 전적으로 호스트 OS의 IME가 담당한다.
- 펌웨어 수준에서 자모 매핑, 한글 조합 매크로, 언어 전용 레이어를 정의하지 않는다.

이 원칙은 다음을 보장한다.

- 운영체제 간 동작 일관성
- 블루투스 및 USB HID 호환성
- 키보드 교체 시 학습 비용 최소화
- ZMK 업데이트에 따른 유지보수 안정성

이 저장소는 IME 대체를 목표로 하지 않는다.

---

## 두벌식과 세벌식의 위치 규정

### 두벌식 (Dubeol-sik, 2-set)

![korean 2set](/docs/images/korean_2set_layout.svg)
두벌식은 기본 기준 레이아웃이다.

- 모든 주요 OS에서 기본 지원
- 별도 소프트웨어 설치 불필요
- QWERTY 기준으로 설계됨

이 저장소의 기본 가정은 다음 조합이다.

- **QWERTY + 두벌식**

이는 공용 PC, 회사 장비, 비프로그래머블 키보드 환경과의 전환 비용을 최소화한다.

---

### 세벌식 (Sebeol-sik, 3-set)

![korean 3set](/docs/images/korean_3set_layout.png)
세벌식은 선택적 고급 입력 방식이다.

- 초성, 중성, 종성 분리 입력
- 연타 정확도와 손가락 부담 감소
- 세벌식 390, 세벌식 최종이 대표적

다음 조건에서 적합하다.

- Colemak, Dvorak 등 비-QWERTY 영문 배열 사용
- 장시간 한국어 타이핑 비중이 높음
- 입력 방식 학습 비용을 감수할 수 있음

세벌식은 OS 설정을 통해서만 활성화한다.

---

## 권장 조합 매트릭스

| 영문 배열 | 한글 배열 | 상태             |
| --------- | --------- | ---------------- |
| QWERTY    | 두벌식    | 기본 기준        |
| Colemak   | 세벌식    | 고급 사용자      |
| QWERTY    | 세벌식    | 가능하나 비권장  |
| Colemak   | 두벌식    | 혼란 가능성 높음 |

Colemak과 두벌식을 조합할 경우, OS는 QWERTY 기준 자모 배치를 기대하므로 물리 키 위치와 인지 모델이 충돌한다. 이 저장소는 해당 조합을 최적화하지 않는다.

---

## 한영 전환 처리

한영 전환은 펌웨어가 아니라 OS 단축키를 호출하는 방식으로만 처리한다.

권장 매핑:

- Windows: Right Alt (한/영 키 스캔코드)
- macOS: Caps Lock 또는 Control + Space

ZMK에서는 이를 일반 키 또는 조합 키로 매핑할 수 있으나, 전환 상태 자체를 추적하지 않는다.

---

## 운영체제별 설정 기준

### Windows

- 한국어 입력기: Microsoft IME
- 자판 설정: 두벌식 또는 세벌식 (390, 최종)
- 별도 IME 없이 사용 가능

### macOS

- 한국어 입력 소스 추가
- 두벌식 또는 세벌식 선택
- Caps Lock 기반 전환 권장

### Linux (IBus, GNOME 기준)

- 입력 소스: Korean (Hangul)
- Hangul Keyboard 설정에서 배열 선택
- 데스크톱 환경에 따라 UI 경로는 다를 수 있음

---

## 신세벌식의 위치

신세벌식은 이 저장소의 **비기본 경로**다.

특성:

- 숫자열 의존도 감소
- 40% 이하 배열에서 구조적으로 유리
- 글자판 3열 내 입력 완결 가능

제약:

- Windows, macOS 기본 IME 미지원
- 서드파티 IME 설치 필요
- 공용 환경, 회사 장비, 모바일 연결 시 사용성 저하

이 저장소는 신세벌식을 위해 펌웨어 레벨 지원을 추가하지 않는다.

---

## 서드파티 IME 참고

- Windows: [날개셋 한글 입력기](http://moogi.new21.org/ngs_download.htm)
- macOS: [구름 입력기](https://gureum.io/)

해당 IME 사용은 전적으로 사용자 책임이며, 저장소는 동작 보장을 하지 않는다.

---

## 명시적 비목표

다음 항목은 이 저장소의 목표가 아니다.

- 펌웨어 기반 한글 조합
- 자모 단위 매크로
- 언어별 레이어 자동 전환
- IME 상태 추적 또는 동기화

한국어 입력은 OS 문제이며, 펌웨어는 이를 방해하지 않는 것이 최적의 설계다.
